Данный репозиторий содержит решение тестового задания для позиции
в Polza Outreach Engine.

Проект состоит из двух независимых частей:
1) Проверка email-доменов (MX-записи)
2) Мини-интеграция с Telegram для отправки сообщений в чат

------------------------------------------------------------
1. Проверка email-доменов (MX check)
------------------------------------------------------------

Скрипт проверяет список email-адресов и определяет статус домена:

- «домен валиден»
- «домен отсутствует»
- «MX-записи отсутствуют или некорректны»

### Файл
mx_check.py

### Зависимости
- Python 3.10+
- dnspython

Установка:
pip install dnspython

### Использование

Проверка одного или нескольких email:
python mx_check.py test@gmail.com fake@nosuchdomain.zzz bad-email

Проверка из файла:
python mx_check.py emails.txt

Формат файла emails.txt:
test@gmail.com
fake@nosuchdomain.zzz
bad-email
